# dotnetcore 2.0.0 Todo WebApi example

## Install

Clone this repo: 

`git clone git@github.com:JesseObrien/dotnettodo.git`

Install the dotnet SDK:

https://www.microsoft.com/net/download/core

Restore:

`cd dotnettodo`
`dotnet restore`

## Run

### Development

Live reloading for builds:

`dotnet watch run`

Regular run:

`dotnet run`
